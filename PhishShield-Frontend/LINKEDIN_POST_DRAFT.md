# LinkedIn Post Draft — PhishShield AI

Built **PhishShield AI**, a full-stack phishing defense project focused on real scam patterns seen in India.

It combines:
- a **React + TypeScript** dashboard
- a **FastAPI** phishing-analysis backend
- **TF-IDF + IndicBERT** based risk scoring
- a **Chrome extension** for browser and Gmail warnings

### What makes it useful
Most phishing demos only output **safe** or **risky**.
PhishShield is designed to explain the verdict with a score, category, and clear risk signals so the user understands **why** the content looks dangerous.

### Current project snapshot
- Accuracy: `97.19%`
- F1 Score: `96.52%`
- Dataset rows: `18,684`
- Stored benchmark report: `2000` scenarios / `2300` executions / `100%` accuracy

It has been a great project for bringing together **AI, security, frontend UX, and browser protection** in one workflow.

Always open to feedback on the product, the detection logic, or the user experience.
